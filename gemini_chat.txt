## 🏗️ La Strategia "Quick & Dirty" (7 giorni)

Per rispettare la scadenza, non scriveremo il codice da zero. Useremo **p5.js** o **Phaser**, librerie JavaScript che permettono di far girare il gioco direttamente nel browser (fondamentale per la condivisione sui social).

### Il Game Design semplificato
1.  **Lo Snake:** Invece di una linea di pixel, lo "Snake" è l'icona del bus 46. Ogni persona raccolta si accoda dietro il bus, allungandolo.
2.  **Il Limite:** Impostiamo un contatore (es. 10 persone). Arrivati a 10, il bus cambia colore (diventa rosso/lampeggiante).
3.  **Il Finale:** Quando raggiungi le coordinate di "Tavernelle" (o dopo il decimo passeggero), il bus si ferma, appare un'animazione di esplosione ("BOOM!") e compare il font con il messaggio: *"Il 46 è pieno. Fartela a piedi non è il massimo. Vota Gulliver."*

## 🚀 Come caricarlo online (Senza avere un sito)

Ecco i passaggi per usare GitHub Pages, che bypassa i filtri dell'università:

1.  **Crea un account** su [GitHub](https://github.com).
2.  **Crea un nuovo "Repository"** chiamato `gioco-46`.
3.  **Carica i file:** Ti servono solo tre file (che possiamo scrivere insieme):
      * `index.html` (lo scheletro)
      * `sketch.js` (il cuore del gioco)
      * `style.css` (per centrare il gioco nello schermo)
4.  **Attiva le Pages:** Vai in *Settings* \> *Pages* \> sotto "Branch" seleziona `main` e clicca *Save*.
5.  **Fatto:** Dopo 1 minuto, il tuo gioco sarà online all'indirizzo che ti darà GitHub.

## 🕹️ Il Codice "Scheletro" (p5.js)

Ecco una base funzionale che puoi testare subito nell'editor online di [p5js.org](https://editor.p5js.org/). Copia e incolla questo codice per vedere il 46 in azione:

```javascript
let bus;
let persone = [];
let passeggeri = 0;
let limitePieno = 10;
let giocoFinito = false;

function setup() {
  createCanvas(400, 600);
  bus = createVector(width / 2, height - 50);
}

function draw() {
  background(220);
  
  if (!giocoFinito) {
    // Disegna il "Bus 46"
    fill(255, 150, 0); // Arancione bus
    rect(bus.x, bus.y, 40, 20);
    fill(0);
    text("46", bus.x + 10, bus.y + 15);

    // Logica movimento (semplificata per test)
    if (keyIsDown(LEFT_ARROW)) bus.x -= 3;
    if (keyIsDown(RIGHT_ARROW)) bus.x += 3;
    bus.y -= 1; // Il bus sale verso Tavernelle

    // Trigger Esplosione (Esempio a coordinate o passeggeri)
    if (bus.y < 100 || passeggeri >= limitePieno) {
      esplosione();
    }
  } else {
    schermataVotaGulliver();
  }
}

function esplosione() {
  giocoFinito = true;
}

function schermataVotaGulliver() {
  textAlign(CENTER);
  textSize(24);
  text("💥 BOOM!", width/2, height/2 - 50);
  textSize(18);
  text("Il 46 è pieno.\nOra te la fai a piedi...", width/2, height/2);
  fill(0, 102, 153);
  textSize(30);
  text("VOTA GULLIVER", width/2, height/2 + 80);
}
```
