<a name="italiano"></a>
## 🇮🇹 Italiano

### Descrizione
**Gulliver 46** è un mini-gioco arcade satirico basato sulla realtà del trasporto pubblico ad Ancona (Italia). Il giocatore veste i panni dell'autista della linea 46, una delle tratte più iconiche e affollate che collega il centro città alle facoltà universitarie di Montedago.

La tua missione è semplice ma ardua: portare quanti più studenti ("sardine") possibile a lezione, evitando di trasformarli in splat di sangue sulla strada e cercando di non far esplodere l'autobus per il sovraccarico!

### Caratteristiche
- **Fisica arcade:** Manovra un bus articolato (quasi) realistico.
- **Sistema di stazioni:** Carica passeggeri alle fermate iconiche di Ancona.
- **Difficoltà crescente:** Più passeggeri carichi, più il bus diventa difficile da gestire.
- **Grafica Premium:** Design moderno con micro-animazioni e sistema particellare.
- **Ottimizzato per Mobile:** Supporto touch e joystick virtuale.

### Tecnologie
- [p5.js](https://p5js.org/) - Rendering e logica di gioco.
- [Vite](https://vitejs.dev/) - Build tool e ambiente di sviluppo.
- JavaScript ES6 (Modulare).

### Come giocare (Locale)
1. Installa le dipendenze:
   ```bash
   npm install
   ```
2. Avvia il server di sviluppo:
   ```bash
   npm run dev
   ```

## 🇬🇧 English

### Description
**Gulliver 46** is a satirical arcade mini-game inspired by the public transport reality in Ancona, Italy. You play as the driver of the line 46, one of the most iconic and crowded routes connecting the city center to the university campus in Montedago.

Your mission is simple but challenging: transport as many students ("sardines") as possible to their classes, avoiding running them over and trying not to push the bus to its breaking point!

### Features
- **Arcade Physics:** Maneuver a (semi) realistic articulated bus.
- **Station System:** Pick up passengers at Ancona's iconic stops.
- **Dynamic Difficulty:** The more passengers you load, the harder the bus is to handle.
- **Premium Graphics:** Modern design with micro-animations and particle systems.
- **Mobile Optimized:** Full touch support and virtual joystick.

### Tech Stack
- [p5.js](https://p5js.org/) - Graphics rendering and logic.
- [Vite](https://vitejs.dev/) - Build tool and dev environment.
- Modular JavaScript ES6.

### How to Play (Locally)
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
